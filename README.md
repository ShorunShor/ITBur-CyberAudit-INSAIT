## Описание проекта
**ITBur-CyberAudit-INSAIT** — это консольный инструмент для автоматического аудита безопасности Linux-систем, разработанный в рамках конкурса «Мы – будущее IT-Бурятии».  

Программа выполняет:
- **Анализ прав доступа** (поиск файлов с правами 777/666, SUID/SGID-битов, конфиденциальных данных в файлах).
- **Сетевой аудит** (сканирование открытых портов, обнаружение опасных сервисов (Telnet, FTP и др.), проверка баннеров и анонимного доступа).
- **Аудит пакетов** (сбор версий ПО, проверка по базе известных уязвимостей CVE).
- **Поиск CTF-флагов** `bit26{...}` в 10+ местах: файловая система, закладки Firefox, история команд, переменные окружения, cron, GRUB, базы данных, systemd journal, HTML-комментарии и зашифрованные подсказки.
- **Проверку привилегий** (анализ sudo, поиск подозрительных процессов).
- **Аудит системных журналов** (неудачные попытки входа, смена паролей).
- **Проверку брандмауэра** (UFW/firewalld).

Результат работы сохраняется в **JSON-отчёт** и выводится в цветном консольном формате.

## Требования к системе
- **Операционная система:** Linux (Kali Linux(рекомендуется), Ubuntu, Debian и другие дистрибутивы на базе Debian).
- **Python:** версия 3.6 или выше.
- **Зависимости:** все модули являются встроенными (os, sys, subprocess, json, re, sqlite3, glob, datetime, collections).  
  *Дополнительно:* для проверки баннеров рекомендуется наличие `netcat` (`nc`), но программа работает и без него.
- **Права:** для доступа к защищённым файлам и журналам требуется запуск с `sudo`.

## Инструкция по установке и запуску
Программа не требует внешних зависимостей (использует стандартные библиотеки Python 3).

 **Скачивание:** Нажмите на файл `2 задание аудитор.zip` в списке файлов выше
 или сам скрипт в расширении auditor.py

1.  **Клонирование/Копирование:** Поместите файл `auditor.py` в домашнюю директорию целевой системы.
2.  **Запуск:** Для доступа к критическим узлам (GRUB, /root, логи) запуск должен производиться от имени суперпользователя:
    ```bash
    sudo python3 auditor.py
    ```
3.  **Результат:** После завершения работы скрипт выведет таблицу в консоль и сохранит детальный отчет в файл `audit_report_[дата]_[время].json`.


*(Примечание:скриншот приложен в архиве с проектом, как и скрипт в расширении .py, и отчёт).*
